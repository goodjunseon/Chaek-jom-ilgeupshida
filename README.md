# 📚 책책책 책좀 읽읍시다! 프로젝트

> **도서 대여 통합 관리 시스템**  
> 사용자는 도서를 조회하고 대여 및 반납할 수 있으며, 관리자는 도서를 관리할 수 있는 웹 기반 도서 시스템입니다.
> Spring Boot와 JPA 기반으로 구현되었으며, Thymeleaf를 통해 SSR을 지원합니다.

---
